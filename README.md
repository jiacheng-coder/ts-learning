# 前言

ts与js最大的区别就是类型系统

我对ts的粗浅理解，通过增加开发者的心智负担，来保证系统的类型安全，提升可维护性

# 使用说明

1. 安装typescript: `npm i typescript -g`
2. terminal: `tsc --watch`实时监测`index.ts`文件的变化并将其编译为`index.js`文件
